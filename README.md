# 干眼、睑板腺功能障碍 相关资料

## 总结

- 及早发现，及早治疗
  - 早期症状
    - 眼干
    - 眼睛白色分泌物
    - 视力模糊
  - 看眼科确诊病因
    - 眼表综合检查（爱尔眼科）
      - 泪膜、睑板腺（oculus睑板腺成像）、etc
    - 简单检查
      - 泪液
